# Riddhi Secure Bank (C++ CLI Project)

A simple command-line banking system built using C++.
This project simulates basic banking operations with file-based data persistence.

## Features

- PIN Registration & Login (3 attempt limit)
- Deposit money
- Withdraw money (with balance validation)
- Balance check
- Persistent storage using text files

## Technologies Used

- C++
- File Handling (fstream)
- Input Validation
- Basic CLI Menu System

## How to Run

1. Compile:
   g++ main.cpp -o bank

2. Run:
   ./bank

## Project Structure

- main.cpp → Core banking logic
- .gitignore → Prevents runtime and compiled files from being pushed

## Concepts Demonstrated

- Functions and modular programming
- Conditional logic and loops
- File I/O handling
- Basic authentication logic
- Error handling and input validation
