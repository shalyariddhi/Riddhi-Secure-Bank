# Riddhi Secure Bank (C++ CLI Project)

A simple command-line banking system built using C++.
Features:
- PIN Registration & Login
- Deposit
- Withdraw
- Balance Check
- File-based data persistence

## Technologies Used
- C++
- File Handling (fstream)

## How to Run
1. Compile:
   g++ main.cpp -o bank
2. Run:
   ./bank
